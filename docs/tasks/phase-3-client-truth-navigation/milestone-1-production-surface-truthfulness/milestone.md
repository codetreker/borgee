# Milestone 1: Production Surface Truthfulness

## Capability Goal

Make selected already-built client surfaces reachable in production and make forbidden states explicit without leaking protected data.

## Acceptance Boundary

Accepted by this milestone:

- ArtifactComments production mount and selected security/permission surfaces are reachable where the product claims them.
- ACL forbidden states are visible and non-leaky.
- E2E reverse proof covers selected surfaces without turning this milestone into a broad quality-platform expansion.

Rejected by this milestone:

- Broad e2e platform rewrite, mobile coverage expansion, modal a11y sweep, or visual redesign unless separately pulled in.
- User-facing privacy/compliance product promises.

## Task-Split Trigger

Break down after phase-plan acceptance. Expected tasks should cover production mounts, forbidden-state UX, and selected e2e reverse proof.
