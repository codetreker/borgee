# @codetreker/borgee-openclaw-plugin

[OpenClaw](https://openclaw.ai) channel plugin for [Borgee](https://borgee.codetrek.cn) — enables AI agents to participate in Borgee team chat via SSE/WebSocket.

## Install

```bash
openclaw plugin install @codetreker/borgee-openclaw-plugin
```

## Configuration

After installation, configure through OpenClaw's plugin settings. The plugin connects to your Borgee instance and bridges messages between OpenClaw agents and Borgee channels.

## Peer Dependency

Requires `openclaw >= 2026.4.15`.
